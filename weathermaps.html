Summary

Visualization


The purpose of this project was to analyze how weather changes as you get closer to the equator. To accompllish this nalysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matpllotlib to plot various aspects of the weathere vs latitude.

Factors we looked at included: temperature, cloudiness, wind speed, humidity. The site provides the sources data and visualizations created as a part of the analysis, as well as explanations and descriptions of any trendsd and correlations witnessed.
